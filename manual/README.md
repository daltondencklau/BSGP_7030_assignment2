BSGP 7030 Assignment 2 Part 1 Manual

The purpose of this assignment is to learn how to create and compare linear regression models in Python and R using jupyter notebooks. We were able to see how two different languages handled the data and fit it to the same linear regression. Using a simplified dataset (like regression.csv) we were able to conceptualize the model and what it was telling us.

We used the libraries Pandas, Matplotlib, Scikit-learn, and ggplot.

Pandas is mainly used for data manipulation, preprocessing, and analysis. It helps us create dataframes that make it easier to clean and analyze datasets.

Matplotlib is a data visualization library where we can create graphs with customizations. I prefer seaborn because I feel like it is much more robust and has more options.

Scikit-learn allows us to create the machine learning models. It contains tools not only for linear regression, but clustering, classification, multi-class classification, model evaluation, data pre-processing, and a lot more. It is one of the most useful libraries I have used throughout my experience building machine learning models.

ggplot in R I am not as familiar with, but it seems comprable to Matplotlib where were can create data visualizations.
